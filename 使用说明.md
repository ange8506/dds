# 打地鼠助手 - 使用说明

## 功能说明

本工具会自动检测游戏中4x4网格内的目标变化：
- ✅ 检测到新目标出现时自动点击
- ❌ 识别到炸弹图像时不点击
- ⚡ 每150毫秒扫描一次，快速响应

## 使用步骤

### 1. 安装APK
将构建好的APK安装到手机上

### 2. 开启无障碍服务
1. 打开手机"设置" -> "辅助功能" / "无障碍"
2. 找到"打地鼠助手"
3. 开启服务权限

### 3. 调整游戏区域（重要）
由于不同手机屏幕尺寸不同，可能需要调整游戏区域坐标：

打开 `WhackMoleAccessibilityService.kt` 文件，找到 `initGameArea()` 函数：

```kotlin
private fun initGameArea() {
    // 调整这些参数以匹配你的游戏界面
    val areaWidth = (screenWidth * 0.85).toInt()  // 游戏区域宽度比例
    val areaHeight = (screenHeight * 0.5).toInt() // 游戏区域高度比例
    val left = (screenWidth - areaWidth) / 2      // 左边距
    val top = (screenHeight * 0.35).toInt()       // 上边距
    
    // ...
}
```

### 4. 开始使用
1. 进入游戏界面
2. 点击悬浮窗的"开始"按钮
3. 工具会自动检测并点击目标
4. 需要停止时点击"停止"按钮

## 工作原理

### 检测逻辑
1. **区域划分**：将游戏区域划分为4x4共16个格子
2. **变化检测**：每150ms扫描一次，检测每个格子是否有新内容出现
3. **炸弹识别**：通过节点描述、文本内容判断是否为炸弹
4. **自动点击**：非炸弹目标立即点击

### 炸弹识别方法
工具会检查以下特征来识别炸弹：
- 节点描述包含"炸弹"、"bomb"关键词
- 节点文本包含"危险"、"爆炸"等关键词
- 节点类名包含"bomb"

## 调试技巧

### 如果工具没有反应
1. **检查无障碍权限**：确保已正确开启
2. **调整游戏区域**：修改 `initGameArea()` 中的参数
3. **查看日志**：使用 `adb logcat` 查看错误信息

### 如果误点炸弹
需要改进炸弹识别逻辑，在 `checkIfBomb()` 函数中添加更多特征：

```kotlin
private fun checkIfBomb(nodes: List<NodeInfo>, cellRect: Rect): Boolean {
    // 添加更多炸弹特征判断
    // 例如：检查图标、颜色、大小等
}
```

### 如果漏点目标
1. **降低扫描间隔**：将 `delay(150)` 改为 `delay(100)`
2. **扩大检测区域**：调整格子边界计算

## 性能优化

- 扫描间隔：150ms（可调整）
- 点击延迟：30ms
- 协程异步处理，不阻塞主线程

## 注意事项

⚠️ **重要提示**
- 本工具仅供学习研究使用
- 请勿用于违反游戏规则的行为
- 不同游戏界面可能需要调整参数
- 首次使用建议先测试调整参数

## 常见问题

**Q: 为什么点击位置不准确？**
A: 需要根据实际游戏界面调整 `initGameArea()` 中的参数。

**Q: 如何提高识别准确率？**
A: 在 `checkIfBomb()` 函数中添加更多炸弹特征判断逻辑。

**Q: 可以用于其他类似游戏吗？**
A: 可以，但需要调整游戏区域和识别逻辑。
